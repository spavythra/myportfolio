# Portfolio Assets

This folder contains images and media files for the portfolio website.

## Required Images

Please replace the following placeholder references with your actual images:

1. **profile.jpg** - Your profile/headshot photo (recommended size: 300x300px or 400x400px)
2. **project1.jpg** - Screenshot or image for your first project (recommended size: 600x400px)
3. **project2.jpg** - Screenshot or image for your second project (recommended size: 600x400px)
4. **project3.jpg** - Screenshot or image for your third project (recommended size: 600x400px)

## Image Guidelines

- Use high-quality images
- Optimize images for web (compress without losing too much quality)
- Recommended formats: JPG for photos, PNG for graphics with transparency
- Keep file sizes under 500KB when possible for faster loading
